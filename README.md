# Hotel Reservation System

A simple, console-based Java application to search, book, and manage hotel rooms.

## Features
- **Room Tiers:** Standard, Deluxe, and Suite rooms.
- **Bookings:** Make and cancel room reservations.
- **Payment Simulation:** Simulated checkout and payment processing.
- **Data Persistence:** Automatically saves and loads data using local text files (`rooms.txt` and `bookings.txt`).

## How to Run
1. Open the project folder in VS Code.
2. Open `HotelSystem.java`.
3. Click **Run** or press **F5**.
